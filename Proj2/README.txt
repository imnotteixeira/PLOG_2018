Para testar o programa para diferentes exemplos, basta selecionar a opção no menu inicial do programa.

Para mais testes, é possivel correr o predicado run(+FirstNumber, +AmountOfNumbers, +Multiplier, -ResultList) 
sendo os 3 primeiros agrumentos os inputs do problema e ResultList a lista de numeros solução.

Auxiliarmente, está disponivel em generate.pl o predicado generate/3, que gera dentro de um intervalo pré definido, inputs aleatórios para o problema
